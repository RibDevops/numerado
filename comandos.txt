Instalar em Modo Offline
Copie o diretório modules para a máquina onde você deseja instalar os módulos.
No terminal, navegue até o diretório modules.
Execute o seguinte comando para instalar os módulos em modo offline:

pip install --no-index --find-links=. <nome do pacote>



pip install --no-index --find-links=modulos -r requirements.txt
